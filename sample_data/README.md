# Sample Data

This folder contains compact representative samples from the AQUINAS contest dataset.

## Included Samples

- `raw_json/`: one small raw JSON sensor record per AQUINAS set
- `event_hdf5/`: one preprocessed event-level HDF5 file per AQUINAS set

These files are provided only for repository structure checks, lightweight I/O tests, and format inspection. They are not intended to reproduce the full challenge results.

Full-scale analysis requires the official contest dataset and generated event-level HDF5 cache.
